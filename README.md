# Mill2Meal - D2C Customer Website

Premium grocery-focused, mobile-first, trust-driven D2C experience built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Utility-first CSS
- **React Router** - Client-side routing
- **Lucide React** - Icons

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Premium luxury design with green/gold color scheme
- Mobile-first responsive design with bottom navigation
- 22+ customer-facing pages
- Product catalog with categories, filters, and sorting
- Shopping cart with quantity management
- Single-page checkout with OTP login
- Order tracking with visual timeline
- Monthly essentials & subscription plans
- Store locator
- Pincode serviceability check
- Global search with suggestions
- WhatsApp ordering integration
